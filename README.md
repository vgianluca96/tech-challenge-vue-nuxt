# Tech Challenge - Frontend

This is the frontend of the Tech Challenge Project.

## Quick setup

* Clone the repo
* run `npm install`
* set the .env file as follows

```bash
BACKEND_URL=<URL of the Gateway app>
```

* run `npm run dev` for local/development environment, or `npm run build && node .output/server/index.mjs` for production build


## Techonologies

* Nestjs to build the app
* MongoDB for data persistence
